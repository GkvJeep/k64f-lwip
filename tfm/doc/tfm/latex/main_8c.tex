\subsection{Detailed Description}
Controls remotely a F\+R\+D\+M-\/\+K64F development board. 

\begin{DoxyAuthor}{Author}
R\+P\+C0027 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
December 2018 
\end{DoxyDate}
\begin{DoxyVersion}{Version}
1.\+0
\end{DoxyVersion}
The code initializes several hardware peripherals of the F\+R\+D\+M-\/\+K64F board and allows to interact with them remotely. The board gains its ability to work remotely through the usage of the lw\+IP library using its T\+C\+P/\+IP stack.
